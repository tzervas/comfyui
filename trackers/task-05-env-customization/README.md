# Task 05: Environment-Based Customization

## Overview
Support dev/staging/prod environments with config overrides.

## Subtasks
1. **Env Files**: Create .env.dev, .env.prod with overrides.
2. **Compose Profiles**: Use profiles to enable/disable services.
3. **Config Overrides**: Allow port, volume, service customizations.
4. **Hugging Face Integration**: Add HF_TOKEN handling for gated models.
5. **Validation**: Ensure configs are valid across environments.
6. **Documentation**: Guide for switching environments.
